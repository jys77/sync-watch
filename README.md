# Sync Watch

## Feature

Users can sync watch a video together, including sync pause, play, and drag the progress bar.

## Installation
1. Server
   ```
   cd server
   npm install
   node index.js
   ```
2. Client
   
   put your video into `/client/src/components/player/`, and name it `video.mp4`.
   ```
   cd client
   npm install
   npm start
   ```
